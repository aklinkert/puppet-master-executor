# puppet-master-executor

Executor service, running jobs for [puppet-master.io](https://puppet-master.io). Puppet-master makes the execution of website interactions
super simple by abstracting the code execution behind a HTTP API, scheduling the job for you in a scalable
manner. For more information please head over to the [puppet-master docs](https://docs.puppet-master.io).


## installation & usage

For detailed information about installation and usage please read the [self hosted section the docs](https://docs.puppet-master.io/#/self_hosted).
